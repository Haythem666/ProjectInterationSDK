# ProjectIntertionSDK

This project contains a collection of SDKs for Android applications.

## Features

### Shopping Cart
A simple shopping cart implementation.

**Modules:**
* **core:** Contains the core business logic for the shopping cart.
* **views:** Contains the UI components for the shopping cart.

### Password Strength Meter
A component to visualize the strength of a password.

**Modules:**
* **core:** Contains the password strength calculation logic.
* **visualizers:** Contains different UI components to visualize password strength.
